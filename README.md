# 🏨 Hotel Management System (SQL Project)

A Hotel Management System database project built using **SQL**.  
This project helps manage hotel operations such as customers, rooms, bookings, payments, and staff details.

---

## 📌 Features
- Manage **Customer details**
- Handle **Room information**
- Store **Booking records**
- Maintain **Payment details**
- Staff/Employee management (if included)
- Easy to import into **MySQL / SQL Server**

---

## 🛠️ Technologies Used
- **SQL**
- MySQL / SQL Server (depending on your usage)

---

## 📂 Project Files
- `HOTEL-MANAGEMENT PROJECT.sql` → Main SQL database file
